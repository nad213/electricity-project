API publique ElecFlow v1

Accès en lecture seule, au format JSON, aux données électriques françaises (source : ODRE). Les puissances sont exprimées en MW.

Documentation interactive (Swagger)

Endpoints

Méthode Endpoint Description
GET /api/v1/meta Plages de dates disponibles et listes de référence (filières, pays).
GET /api/v1/consommation Consommation (puissance, MW). Paramètres : start_date, end_date.
GET /api/v1/production Production par filière (MW). Paramètres : start_date, end_date, filiere.
GET /api/v1/echanges Échanges transfrontaliers (MW). Paramètres : start_date, end_date, pays.
GET /api/v1/parc-installe Parc installé mensuel éolien/solaire (MW).

Paramètres communs

  • start_date / end_date — format AAAA-MM-JJ. Plage maximale : 366 jours.
  • filiere — clé de filière (ex. nucleaire, eolien, solaire…). Voir /meta.
  • pays — clé de frontière (ex. ech_physiques…). Voir /meta.

Exemples

Métadonnées :

curl "{{ api_base }}/meta"

Consommation sur une période :

curl "{{ api_base }}/consommation?start_date=2024-01-01&end_date=2024-01-07"

Production solaire :

curl "{{ api_base }}/production?start_date=2024-06-01&end_date=2024-06-07&filiere=solaire"

Échanges physiques :

curl "{{ api_base }}/echanges?start_date=2024-01-01&end_date=2024-01-07"
