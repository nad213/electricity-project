Démarrage rapide

Chaque requête porte l'en-tête Authorization: Bearer <clé>. Exportez votre clé une fois, puis appelez un endpoint : 

export ELF_KEY="votre_clé"

curl -H "Authorization: Bearer $ELF_KEY" \
  "{{ api_base }}/courbe_prod?debut=2024-06-01&fin=2024-06-07&filiere=solaire"

… ou une agrégation d'énergie par mois (GWh) :

curl -H "Authorization: Bearer $ELF_KEY" \
  "{{ api_base }}/energie_echange?debut=2024-01-01&fin=2024-12-31&pays=total"
